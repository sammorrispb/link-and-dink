# The Session Design Framework

*The Operating Principle says: have a plan, and hold it loosely. This framework is the "have a plan" half — a repeatable design process you run* before *you ever get to the court. The "hold it loosely" half is [Reading the Room](reading-the-room).*

---

## Why design beats improv

You cannot [design an environment](../creed/the-coach-as-environment-designer) you haven't thought about. A coach who shows up with a hopper and improvises is, at best, running other people's drills from memory — and at worst, the basket-of-balls gig Coach Up exists to replace. Designing a session in advance is a 15-minute habit that separates a coach from a ball-feeder.

It is also *respect*. An adult gave you a protected, paid hour of their week. Walking in with a design says you took that seriously.

---

## The five-step design process

Run this before every session. It's quick once it's a habit.

### 1. Pick one focus

One. A session has a single focus — almost always one block from [the Adult Skill Progression](the-adult-skill-progression). "Today is the third-shot decision." Not "third shots and dinks and footwork." A session that's about three things is about nothing; the players leave unable to name what they worked on, which fails [the Pillars' audit](the-coach-up-pillars).

### 2. Know your players

Who's coming, what level, what's each person's running focus from last time, any bodies to be careful with, any egos to manage. For a 1:1 this is deep; for a clinic it's broad. Either way: you design for *these* people, not a generic player. If you genuinely don't know them yet (a first session, a new clinic), your design's job is partly to *find out* — build in early reps that show you where they are.

### 3. Choose the games that rep the focus

Now go to [The Constraint Design Toolkit](the-constraint-design-toolkit) and pick the constraints and games that make today's focus the path of least resistance. For each one, you must be able to say in a sentence what it reps. If you can't, it's not in the session.

### 4. Sequence easy to hard

Order the blocks so each one is a step up from the last. Start where the players can succeed — engineering early success is real motor-learning practice, not just kindness ([Research Foundations](../creed/research-foundations), OPTIMAL theory). Then add chaos: fed ball before live ball, bigger target before smaller, cooperative before competitive. The session should *climb*.

### 5. Script the frame and the off-ramps

Two things to write down, literally:
- **The frame** — the one or two sentences you'll open with that tell the players *why* today's focus matters. Adults need the why before the how. Script it so it's tight.
- **The off-ramps** — for each game, your loosening move if the room collapses (a bigger target, a fed ball, a lower count). Decide these now, calm, not later, scrambling.

## The default Coach Up session arc

Most sessions — a lesson, a small group, a clinic block — fit this shape. It's the adult cousin of the youth session arc in the seed material: same logic, adult pacing.

```
┌────────────────────────────────────────────────────────┐
│  WARM-UP RALLY        Players warm themselves up with    │
│  (~5–8 min)           live dinking / soft rallies.       │
│                       You greet, watch, take a read.     │
├────────────────────────────────────────────────────────┤
│  THE FRAME            You name today's focus and the     │
│  (~2 min)             why. Short. The only real "talk"   │
│                       block in the session.              │
├────────────────────────────────────────────────────────┤
│  SKILL BLOCKS         The bulk. Constraint games that    │
│  (the bulk)           rep the focus, sequenced easy →     │
│                       hard. You coach to the Pillars.    │
├────────────────────────────────────────────────────────┤
│  LIVE APPLICATION     A real game or game-like format    │
│  (~10–15 min)         where the focus has to survive     │
│                       full-speed play.                   │
├────────────────────────────────────────────────────────┤
│  ONE TAKEAWAY EACH    Every player leaves able to say    │
│  (~2–3 min)           what they worked on and one thing  │
│                       to keep doing. Non-negotiable.      │
└────────────────────────────────────────────────────────┘
```

Notes on the arc:
- **The warm-up is working.** Players warming themselves up with live dinks is a rep of the soft game *and* your first read of the room. Don't replace it with a coach-led calisthenics block.
- **The frame is the only monologue.** Everything after it is players doing, you coaching alongside. If you're talking in long stretches during the skill blocks, the design failed.
- **Live application is non-optional.** A focus that never has to survive a real game hasn't been learned — it's been rehearsed. The constraint comes off; the game tests it.
- **One takeaway each closes the loop.** This is the [Pillar audit](the-coach-up-pillars) made into a habit. If a player can't name their takeaway, you were active but you didn't teach.

A 1:1 compresses this arc; a 90-minute clinic runs it with more skill blocks or two focuses stacked. The shape holds. (The format-specific versions live in [Systems](../systems/structure-a-1-1-lesson).)

## The design, on one index card

If you wrote your whole session plan on a card, it would be six lines:

```
FOCUS:      ____________________   (one block)
PLAYERS:    ____________________   (who, level, running focus)
FRAME:      ____________________   (the why, scripted)
BLOCKS:     1. ______  2. ______  3. ______   (easy → hard, each reps the focus)
LIVE GAME:  ____________________   (the focus, full speed)
OFF-RAMPS:  ____________________   (the loosening move for each block)
```

That card is the difference between a coach and a hopper of balls. Fill it out before you leave the house.

> Pick one focus. Know who's coming. Choose the games that rep it. Climb easy to hard. Script the why and the off-ramps. Then go, and hold it loosely.
