# Game-Based Teaching

*The default mode of a Coach Up session. Not "no drills ever" — "the game is the teacher, by default."*

**Definition.** A teaching approach that uses modified, game-like activities — rather than isolated, repetitive drills — as the primary vehicle for learning. Players develop tactical awareness, decision-making, and technique together, in the context where they'll actually be used.

**Why it matters.** A skill drilled in isolation often doesn't survive contact with a real game, because the game adds reading, deciding, and pressure that the drill stripped out. Game-based teaching keeps those in. It's also simply more *engaging* — which matters enormously for adult retention.

**Coaching it (you, the coach).** This descends from Teaching Games for Understanding (Bunker & Thorpe — see [Research Foundations](../creed/research-foundations)). It pairs tightly with [constraint-led coaching](term-constraint-led-coaching): the constraint is what shapes the game toward the skill. It does *not* ban drills — a fed-ball block to find a swing is sometimes the right [off-ramp](../frameworks/the-constraint-design-toolkit). The rule is that the game is the default and the drill is the exception, not the reverse.

> Teach the game with a game — drills isolate, games integrate.
