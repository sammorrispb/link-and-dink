# Return of Serve

*Most adults treat the return as just "get it back." Coached well, it's the single most valuable positional habit in the game.*

**Definition.** The shot that returns the serve. By the two-bounce rule it must be allowed to bounce, then it's played back deep into the opponents' court — ideally landing them well behind their baseline.

**Why it matters.** Here's the leverage: after the returner hits, *both members of the returning team should advance to the kitchen line.* A **deep** return gives them the time to do that. A short return doesn't — it pulls the returner forward into no-man's-land and hands the serving team an easy third. The return's depth literally buys the returning team the net.

**Coaching it (adults).** Frame the return as a *positioning tool*, not a weapon — the goal is "deep, then go," not "winner." The most common adult error is admiring the return instead of moving; coach the move *off* the shot. Pair it with the serve as block 4 of [the Adult Skill Progression](../frameworks/the-adult-skill-progression).

> Return deep, then go — the depth is what buys you the net.
