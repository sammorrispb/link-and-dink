# Error Detection

*The skill underneath diagnosis. And, eventually, a skill you hand to the player.*

**Definition.** The ability to correctly identify *what actually went wrong* in a performance — distinguishing the root cause from the visible symptom. For a coach, it's the diagnostic skill that makes teaching accurate. For a player, learning *self*-error-detection is a milestone: it's the point where they can adjust without the coach.

**Why it matters.** Every wrong correction starts with a wrong error-detection — the coach saw the swing and missed that the real problem was no intent. And [Creed belief #9](../creed/the-coach-up-creed) — the job is to become unnecessary — is partly about *transferring* error detection to the player.

**Coaching it (you, the coach).** Your error-detection tool is [the Hierarchy of Assessment](../frameworks/hierarchy-of-assessment) — walking the levels in order is exactly an error-detection discipline. To build it in *players*, ask before you tell: "what do you think happened there?" A player who can answer that accurately is a player learning to coach themselves.

> See the cause, not the symptom — and teach the player to see it too.
